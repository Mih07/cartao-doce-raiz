# Cartão de Visita Digital

Cartão de visita digital desenvolvido em HTML, com foco em uso mobile e botão funcional para contato direto via WhatsApp.

## Funcionalidades
- Layout responsivo
- Botão clicável para WhatsApp
- Design simples e objetivo
- Uso como link na bio ou QR Code

## Tecnologias utilizadas
- HTML
- CSS

## Objetivo
Projeto criado para fins de estudo e portfólio, demonstrando a criação de um cartão digital funcional para pequenos negócios.
